Copy a file

Usage: app copy [options] <source> [destination] [...more-destinations]

Positional parameters:
  <source>               The source file
  [destination]          The destination file
  [...more-destinations] The destination files

Options:
  -h, --help      Show help
  -f, --force     Force overwrite
  -r, --recursive Copy recursively
      --dry-run   Dry run
